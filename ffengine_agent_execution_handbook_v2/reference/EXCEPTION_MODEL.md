# Exception Model

## Temel Sınıflar
```python
class FFEngineError(Exception): ...
class ConfigError(FFEngineError): ...
class ValidationError(FFEngineError): ...
class ConnectionError(FFEngineError): ...
class DialectError(FFEngineError): ...
class MappingError(FFEngineError): ...
class EngineError(FFEngineError): ...
class DeliveryPolicyError(FFEngineError): ...
class CheckpointError(FFEngineError): ...
```

## Kurallar
- Dışarıya `ValueError`, `RuntimeError` ve sürücü özel exception'lar çıplak taşınmaz.
- Hata, domain exception'a wrap edilmelidir.
- Log kaydı ile exception türü tutarlı olmalıdır.
- Community'de retry task/chunk seviyesinde; Enterprise'da message-level retry yalnızca delivery policy ile yapılır.
