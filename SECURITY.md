# Security Policy

## Supported Scope

Security issues for the public FFEngine Community repository should be reported
privately and must not be disclosed through public issues.

## Reporting

Until a dedicated security mailbox or GitHub security workflow is configured,
report vulnerabilities privately to:

- `caglar.sahin@ffengine.com`
- `burcu.sahin@ffengine.com`

Include:

- affected version or commit
- reproduction steps
- impact summary
- proposed mitigation if known

## Response Policy

- Acknowledge receipt
- Reproduce and triage privately
- Fix in a protected branch
- Publish remediation notes after release when appropriate
